## 5. Booking and Rental Process

### 5.1 Creating Bookings

**Booking Workflow:**

1. **Order Creation:**
   - Buyer creates a new order with a unique order name
   - Selects equipment (SKUs) from owner's inventory
   - Specifies quantity for each item (1-20 units)
   - Adds crew members if required (1-10 members)
   - Sets rental start and end dates

2. **Availability Check:**
   - System checks equipment availability for selected dates
   - Displays pricing breakdown:
     - Base rental amount
     - Crew charges (if applicable)
     - Coupon discount (if applicable)
     - Advanced payment amount
     - Final payable amount

3. **Booking Request:**
   - Buyer submits booking request
   - Booking status: "Pending Request"

4. **Owner Approval:**
   - Owner receives notification of new booking request
   - Owner reviews booking details
   - Owner can accept or reject with reason
   - Acceptance changes status to "Pending Approval"

5. **Booking Confirmation:**
   - Upon final confirmation, status changes to "Pickup Due"
   - Both parties receive confirmation notification
   - Rental dates are reserved

### 5.2 Pickup Process

**Mandatory Pickup Verification:**

1. **Request Pickup OTP:**
   - Owner requests pickup OTP via the App
   - System sends SMS OTP to buyer's registered phone number
   - OTP valid for 10 minutes

2. **Equipment Handover:**
   - Owner hands over equipment to buyer
   - Buyer provides OTP to owner

3. **OTP Verification:**
   - Owner enters OTP in the App
   - System verifies OTP
   - Upon successful verification, booking status changes to "In Progress"

4. **Documentation:**
   - Owner uploads photos of equipment condition
   - Owner uploads challan document
   - Photos serve as proof of equipment condition at pickup

**Failure to Complete Pickup:**
- If pickup OTP is not verified within agreed timeframe, booking may be cancelled
- Cancellation fees may apply as per Section 5.6

### 5.3 Rental Period

**During Rental:**

- Booking status remains "In Progress"
- Buyer is responsible for equipment safety and maintenance
- Owner cannot rent the same equipment to others
- Rental charges accrue as per agreed terms
- Extensions must be requested and approved before end date

**Extensions:**
- Buyer must request extension at least 24 hours before end date
- Owner must approve extension and new pricing
- Extension creates a modified booking with new end date

### 5.4 Return Process

**Mandatory Return Verification:**

1. **Request Return OTP:**
   - Owner requests return OTP via the App
   - System sends SMS OTP to buyer's registered phone number
   - OTP valid for 10 minutes

2. **Equipment Inspection:**
   - Owner inspects equipment condition
   - Buyer provides OTP to owner
   - Any damage or issues are documented immediately

3. **OTP Verification:**
   - Owner enters OTP in the App
   - System verifies OTP
   - Upon successful verification, booking status changes to "Return Confirmed"

4. **Documentation:**
   - Owner uploads photos of equipment condition after return
   - Owner uploads return challan document
   - Photos serve as proof of equipment condition at return

5. **Booking Completion:**
   - After all documentation is complete, status changes to "Completed"
   - Booking moves to history
   - Settlement process can now begin

**Late Returns:**
- Late return charges apply as per Section 6.3
- Equipment must be returned regardless of late fees dispute

### 5.5 Booking Status Lifecycle

```
1. Draft → Initial order creation
2. Pending Request → Awaiting owner review
3. Pending Approval → Owner has accepted, awaiting final confirmation
4. Pickup Due → Confirmed booking, awaiting pickup
5. Pickup Confirmed → OTP verified, equipment handed over
6. In Progress → Active rental period
7. Return Pending → Return date reached, awaiting return
8. Return Confirmed → Equipment returned and verified
9. Completed → Booking fully closed
```

**Additional Statuses:**
- **Cancelled:** Booking cancelled by either party
- **Rejected:** Owner rejected booking request
- **Failed:** Technical failure or payment failure

### 5.6 Cancellation Policy

**Buyer-Initiated Cancellation:**

- **Before Confirmation:** Free cancellation, no charges
- **After Confirmation, 24+ hours before pickup:** 20% cancellation fee
- **Within 24 hours of pickup:** 50% cancellation fee
- **After pickup (no-show):** 100% rental charges apply
- **During rental period:** No cancellation allowed; full rental charges apply

**Owner-Initiated Cancellation:**

- **Before Confirmation:** Free cancellation with valid reason
- **After Confirmation:** Not permitted except in cases of:
  - Equipment breakdown or damage
  - Force majeure events (natural disasters, etc.)
  - Buyer's violation of Terms
- **Unjustified Cancellation:** Owner may face penalties, including:
  - Account suspension
  - Negative rating impact
  - Compensation to buyer for alternative equipment costs

**Dispute Resolution:**
- Cancellation disputes may be escalated to support
- We will review evidence and make final determination
- Our decision is binding unless challenged through arbitration

### 5.7 No-Show Policy

**Buyer No-Show:**
- Failure to pick up equipment within 2 hours of agreed time without notice
- Booking automatically cancelled
- 100% rental charges apply
- Equipment becomes available for other bookings

**Owner No-Show:**
